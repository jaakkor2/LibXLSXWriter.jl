# LibXLSXWriter

Julia wrapper for [libxlsxwriter_jll](https://github.com/JuliaBinaryWrappers/libxlsxwriter_jll.jl). [libxlsxwriter](https://github.com/jmcnamara/libxlsxwriter) is a C library for creating Excel XLSX files.
